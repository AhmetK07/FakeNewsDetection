# Hicivli Haber Tespiti (Türkçe)
Bir mezuniyet projesi adı altında yoldaşlarımla birlikte beraber açtığımız ve tamamladığımız Makine öğrenmesi teknikleriyle çalışan bir model girişimidir. Amacımız başlıkta belirtildiği gibi Türkiye de gerek gerçek ile alaycı sahte haberleri ayırt etmekte yardımcı olacak bir Makine öğrenmesi modeli yaratmaktır.
